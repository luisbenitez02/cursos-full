DOM - Document Object Model & Scripting

Representa todo el contenido del documento

Tiene 'nodos' que pueden ser etiquetas HTML, texto, atributos de los elementos

Trabajaremos sobre el HTML/JS en la carpeta 'universidad'

Query Selector

Retorna el emento seleccionado con #id o .class, tambien permite seleccionar imagenes, elementos HTML

Sin embargo solo retorna el primer elemento aunque este involucre a varios

En general QuerySelector te permite usar todos los selectores disponibles en CSS

Seleccionando multiples Elementos

getElementsByClassName & getElementsByTagName

Seleccionar varios elementos de acuerdo a su .class y de acuerdo a su etiqueta HTML (respectivamente)

QuerySelectorAll

Permite obtener una lista de resultados
