For

Sintaxis para arreglos

for auto in automovil, esto permite usar el ciclo for especialmente en arreglos

Ciclo While

Debemos incrementar dentro del while, podemos hacerlo antes o despues de ejecutar nuestro codigo de interes

          while (i<11) {
               console.log("Sumando unito: " + i);
               i++;
          }
