Operaciones con Strings y demas

.length

Te permite conocer extension del string

.toUpperCase() .toLowerCase()

Ponen el texto en mayuscula o miniscula respectivamente

.indexOf()

Busca el texto en el string, devuelve la posicion donde inicia

.substring()

Muestra la cadena desde los valores dados en el array String

.slice()

De atras hacia adelante

.split()

Corta porciones de texto, busca lo que ingreses y lo va cortando

Replace()

Busca lo que incluyas y lo reemplaza

includes()

Revisa si incluye la cadena y devuelve true o false

Repeat(n)

Repite n veces lo que escribes
